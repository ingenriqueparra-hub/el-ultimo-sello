# SKILLS.md
# Skills necesarias para desarrollar **El Último Sello**

Este documento define las capacidades que el proyecto necesita, tanto para personas como para agentes IA.

---

## 1. Game Design

### Objetivo
Diseñar un juego claro, tenso y rejugable basado en inspección, decisión y consecuencias.

### Capacidades necesarias

- Diseñar loops jugables.
- Balancear dificultad.
- Crear reglas progresivas.
- Diseñar sistemas de penalización.
- Crear decisiones con ambigüedad moral.
- Evitar que el juego se vuelva repetitivo.
- Convertir burocracia en tensión.

### Entregables

- Core loop.
- Tabla de reglas por día.
- Diseño de documentos.
- Matriz de inconsistencias.
- Diseño de decisiones.
- Sistema de scoring y consecuencias.

## 11. Skill crítica: reducir alcance

Cada vez que una idea nueva aparezca, se debe preguntar:

1. ¿Hace más claro el loop principal?
2. ¿Aumenta la tensión o solo agrega complejidad?
3. ¿Puede probarse en el MVP?
4. ¿Requiere arte, sistemas o narrativa adicional?
5. ¿Puede dejarse para una versión posterior?

Si no fortalece el MVP, va al backlog futuro.
