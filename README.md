# El Último Sello — Documentación inicial

Este paquete contiene la documentación base para iniciar el desarrollo de un videojuego indie para Steam: un simulador narrativo de inspección migratoria en un imperio galáctico oscuro.

## Archivos incluidos

- `PROJECT.md`: visión general, alcance, principios y riesgos.
- `AGENTS.md`: instrucciones para trabajar con agentes IA.
- `SKILLS.md`: capacidades necesarias del proyecto.
- `GAME_DESIGN.md`: diseño jugable, loop, documentos y decisiones.
- `NARRATIVE_BIBLE.md`: mundo, facciones, tono y personajes.
- `MVP_BACKLOG.md`: backlog inicial y alcance cerrado del MVP.
- `TECHNICAL_SPEC.md`: arquitectura técnica sugerida.
- `STEAM_MARKETING.md`: posicionamiento, pitch y marketing para Steam.
- `IP_SAFETY.md`: guía para evitar riesgos de propiedad intelectual.

## Nombre provisional

**El Último Sello**

## Concepto

Un thriller burocrático de ciencia ficción oscura donde el jugador revisa documentos, detecta anomalías, interroga solicitantes y decide quién puede entrar a una ciudad-colmena al borde del colapso.

## Motor recomendado

Godot 4.

## Primera meta

Construir un MVP con:

- 1 día jugable;
- 10 solicitantes;
- 3 documentos;
- 1 herramienta de escáner;
- 3 decisiones;
- 1 reporte final;
- 1 consecuencia narrativa.
